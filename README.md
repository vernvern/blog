# blog

## 项目结构

```
.
├── blog
│   ├── api          #  接口
│   ├── models       #  数据类
│   ├── modules      #  业务代码
│   ├── static       #  静态文件
│   ├── templates    #  模版
│   ├── server.py    #  项目程序入口
│   ├── settings.py  #  配置
│   ├── toolkit      #  与业务无关的封装
│   └── views        #  视图
├── docs             #  文档
├── manage.py        #  控制台( 打开项目、跑单元测试、跑脚本、etc.)
├── scripts          #  脚本
├── requirements.txt #  依赖
└── tests            #  单元测试

```
